O Jogo da Vida não é um jogo de computador típico. É um autômato celular e foi inventado pelo matemático John Conway, de Cambridge.

Este jogo se tornou amplamente conhecido quando foi mencionado em um artigo publicado pela Scientific American em 1970. Consiste em uma grade de células que, com base em algumas regras matemáticas, podem viver, morrer ou se multiplicar. Dependendo das condições iniciais, as células formam vários padrões ao longo do jogo.
